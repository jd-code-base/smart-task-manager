# Smart-Task-Manager

A modern, full-featured task management application built with:

- **Frontend**: Vite + React + Tailwind CSS  
- **Backend**: Node.js + Express  
- **Database**: MongoDB  
- **Cache/OTP/Rate Limiting**: Redis  
- **Email Services**: Nodemailer
